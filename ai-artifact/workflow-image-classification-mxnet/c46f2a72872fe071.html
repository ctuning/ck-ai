Pre-requisites:
  • CUDA compatible device
  • CUDA 8.x library and compiler (will be automatically detected by the CK)
  • Python 2.7+ or 3.4+
  • Collective Knowledge
  • PIP
  • Other Python dependencies should be installed by the CK
Installation:
ck pull repo:ck-mxnet
ck install package --tags=lib,mxnet-cuda,vprebuilt,v0.11.0,vcuda-80
ck install package --tags=mxnetmodel
Usage example (image classification):
ck run program:mxnet
