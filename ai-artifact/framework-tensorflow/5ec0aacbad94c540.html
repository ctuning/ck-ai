Pre-requisites:
  • CUDA compatible device
  • CUDA 9.x library and compiler (will be automatically detected by the CK)
  • Possibly cuDNN 6+
  • Python 3.6
  • Collective Knowledge
  • PIP
Installation:
$ pip install --upgrade pip
$ pip install protobuf easydict joblib image numpy scipy

pip install ck
ck pull repo:ck-tensorflow
ck install package --tags=lib,tensorflow,vcuda,v1.4.0
Usage example (image classification):
ck run program:tensorflow
Crowd-benchmarking (see paper):
ck crowdbench tensorflow --user={your email or ID to acknowledge contributions} (--env.BATCH_SIZE=128 --env.NUM_BATCHES=100)
